CEO 现金流驾驶舱 V3 数据验证基线 ​

目标读者与使用方式 ​

本组材料面向产品、CFO/财务和研发。阅读后应能够填写样本模板、实现导入和计算链路，并对指标逐项核对。

交付物 ​

  1. 《V3 数据库表结构与字段字典》：定义标准业务模型、字段、约束及待确认事项。
  2. 《V3 计算规则与 CFO 确认表》：定义指标、四张决策卡、风险与任务规则。
  3. 《V3 API 设计》：定义导入、预览、发布、驾驶舱、风险、任务和试算接口。
  4. 《V3 财务样本数据模板 V0.1》：财务填写的账户余额、应收款、实际回款、计划支出模板。
  5. 《30 / 60 / 90 天现金流预测模块 MVP 技术方案初稿》：定义未来 90 天逐日现金流、窗口最大缺口、安全线与恢复日期的计算口径。
  6. 《回款风险 Top 3 模块 MVP 技术方案初稿》：定义应收未回金额、临期与逾期、风险等级及 Top 3 排序口径。
  7. 《付款建议 Top 3 模块 MVP 技术方案初稿》：定义付款资格、优先级排序、逐笔现金流试算、建议结果持久化和 AI 按需解释。
  8. 《今日老板要拍板模块 MVP 技术方案初稿》：定义回款与付款事件升级边界、老板处理选项、决策留痕和 AI 按需解释。
  9. 《MVP 后端待确认口径清单》：集中记录暂行规则、待确认事项及其影响，不阻塞确定性能力开发。
  10. 《V3 MVP 后端主要实现说明》：按模块说明当前后端实现、计算逻辑、接口对接优先级和待确认边界。
  11. 《V3 MVP 后端数据流图》：用数据流和时序图说明 Excel、标准表、规则引擎、老板拍板、AI 解释和前端页面之间的关系。
